## ðŸ“– Overview
This project was executed as part of **ITEC 380: Managing IT Projects**.  
The initiative aimed to **replace outdated printers, scanners, multi-function devices, and projectors across campus** in compliance with a new state contract.  

The project was managed under a strict **60-day timeline** and a **$393,000 cost baseline**, requiring integration of scope, schedule, cost, communication, HR, risk, and stakeholder management.
